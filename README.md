# Welcome to My Portfolio

## To get started
```
npm install
yarn install
```

## To start development server
```
npm run dev 
yarn run dev 
```

## To start production server
```
npm run build
yarn run build 
```

## Linting
```
npm run lint
yarn run lint
```

This website is keyboard accessible and designed specifically for screenreaders to have the best experience. 

The tech stack: 
- Next.js v13.4
- Tailwind CSS v3.3.
- React v18
- Headless UI v1.7
- MDX v2.1
